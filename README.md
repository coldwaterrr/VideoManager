# VideoSorter 视频管理器

一个基于 Electron + React + TypeScript 的本地视频映射管理桌面应用。

**核心特性：**
- 🎬 **智能扫描**：递归扫描目录，支持多种视频格式，可配置文件大小和格式筛选器
- 📁 **虚拟分类**：创建虚拟文件夹管理视频，无需移动物理文件
- 👆 **悬停预览**：鼠标悬停即可预览视频片段
- 🖼️ **随机封面**：自动从视频 10%-50% 位置截取封面
- 🔍 **实时搜索**：按名称或路径快速过滤
- 📄 **分页浏览**：每页 12 个视频，浏览更流畅
- 🗑️ **批量操作**：支持批量删除视频（搜索过滤后也可批量）
- 🖥️ **中文界面**：完整的本地化支持

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动应用

**推荐方式：**
```bash
# 双击 start.bat 启动
```

**开发模式：**
```bash
npm run dev
```
这将同时启动 Vite 开发服务器和 Electron 应用窗口。

### 3. 构建应用

```bash
npm run build
```
构建产物位于 `dist/` 和 `dist-electron/` 目录。

## 使用指南

### 扫描视频

1. 点击顶部「筛选器」按钮展开筛选器面板（可选）
   - 设置最小/最大文件大小（MB）
   - 选择要扫描的视频格式
2. 点击「扫描目录」按钮
3. 选择包含视频文件的文件夹
4. 应用会递归扫描并显示进度，找到的视频会添加到数据库

### 管理虚拟文件夹

- **创建文件夹**：在左侧边栏输入文件夹名称并点击「创建虚拟文件夹」
- **添加视频**：点击视频卡片上的文件夹按钮，视频会被添加到该文件夹
- **移除视频**：再次点击同一文件夹按钮即可取消关联

### 批量删除视频

1. 勾选要删除的视频卡片左上角的复选框
2. 或点击「全选」选中当前页所有视频
3. 点击「删除选中」按钮
4. 确认后删除（同时删除数据库记录）

**注意：** 删除操作不可撤销

### 搜索与过滤

- 在顶部搜索框输入关键词，实时过滤视频
- 搜索结果支持批量删除和播放

### 视频预览与封面

- **悬停预览**：鼠标移动到视频卡片上，500ms 后自动播放视频片段
- **随机封面**：系统自动从视频 10%-50% 位置截取封面
- **双击打开**：双击卡片用系统默认播放器打开视频

### 系统文件夹

- **全部视频**：显示所有视频（无论是否分类）
- **未分类**：仅显示尚未加入任何虚拟文件夹的视频
- **自定义文件夹**：用户创建的虚拟分类文件夹

## 功能说明

### 扫描筛选器

在扫描前可以设置筛选条件：
- **文件大小**：最小/最大大小（MB），0 表示不限制
- **文件格式**：只扫描选中的格式（mp4, mkv, avi, mov, wmv）

### 批量操作

- 支持批量删除视频
- 删除时会同时清理关联表数据
- 搜索过滤后仍可批量操作

### 分页显示

- 每页显示 12 个视频 cards
- 实时显示当前页/总页数
- 支持跳转到首页、末页、上一页、下一页

### 清理旧格式

如果之前扫描过不支持的格式（如 webm, flv 等），可以在左侧边栏底部点击「清理旧格式记录」删除这些记录。

## 技术栈

- **框架**: Electron 30 + React 19 + TypeScript 5.9
- **构建**: Vite 7 + electron-vite
- **样式**: Tailwind CSS 4
- **数据库**: SQLite (sql.js)
- **图标**: Lucide React
- **UI 风格**: Shadcn UI

## 数据库结构

### videos 表
| 字段 | 说明 |
|------|------|
| id | 主键 |
| absolute_path | 视频绝对路径（唯一） |
| file_name | 文件名 |
| file_size | 文件大小（字节） |
| duration_seconds | 时长（秒，暂未解析） |
| modified_at | 修改时间 |
| created_at | 创建时间 |
| updated_at | 更新时间 |

### virtual_folders 表
| 字段 | 说明 |
|------|------|
| id | 主键 |
| name | 文件夹名称 |
| created_at | 创建时间 |
| updated_at | 更新时间 |

### virtual_folder_videos 表
| 字段 | 说明 |
|------|------|
| id | 主键 |
| virtual_folder_id | 文件夹 ID |
| video_id | 视频 ID |
| created_at | 关联时间 |

## 注意事项

- **预览模式**：在浏览器中运行会显示模拟数据，只有 Electron 窗口才支持实际操作
- **封面生成**：首次加载视频封面需要读取视频元数据，可能需要几秒钟
- **悬停预览**：为避免性能问题，设置 500ms 延迟才开始播放
- **文件路径**：视频文件路径必须可访问，否则无法预览
